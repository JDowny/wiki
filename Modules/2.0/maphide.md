## Description

Replace the output of `/MAP` and `/LINKS` with a URL.

## Configuration Tags

`<security maphide="http://linkto.a/site">`

Attribute | Type | Description
--------- | ---- | -----------
maphide | string | The URL to replace `/MAP` and `/LINKS` with.
